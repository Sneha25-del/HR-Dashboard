# HR-Dashboard
 HR Analytics Dashboard | Power BI
This interactive HR Analytics Dashboard is built using Power BI to help HR teams monitor workforce performance, employee demographics, and attrition trends through clear visualizations and KPIs.

#Project Overview
The dashboard transforms raw HR data into meaningful insights, enabling organizations to make informed decisions regarding employee retention, workforce planning, and HR strategy.

#Key Features
👥 Total Employees & Active Employees KPI Cards
📉 Employee Attrition Count & Attrition Rate Analysis
🎂 Average Employee Age & Experience
🏢 Department-wise Employee Distribution
💰 Attrition by Salary Slab
👨‍💼 Job Role & Satisfaction Analysis
👩‍💼 Gender-wise Attrition Breakdown
📊 Age Group Distribution
📈 Experience Trend Analysis
🎛️ Interactive Department Filter (Slicer)

#Tools & Technologies
Power BI
Microsoft Excel
Power Query
DAX (Data Analysis Expressions)
Data Modeling

#Dashboard Insights
Monitor employee workforce statistics.
Identify departments with higher attrition.
Analyze salary and experience trends.
Understand employee demographics.
Support HR decision-making with interactive visual reports.
